# Attachments de runbooks

Carpeta para imágenes de referencia usadas por los runbooks de
`vault/runbooks/`.

## Convenciones

- Nombre en kebab-case, con el defecto y una etiqueta corta:
  `moho-verde-01.jpg`, `moho-negro-01.jpg`, `moho-blanco-deseado-01.jpg`,
  `case-hardening-corte-01.jpg`.
- Formato preferido: `.jpg` para foto, `.png` para diagramas.
- Fondo neutro (papel blanco / madera clara), luz difusa, y **regla o
  moneda** en el encuadre para dar escala.
- Enfoque nítido sobre la lesión / colonia; evitar sombras duras.
- Insertar en el runbook con sintaxis Obsidian:
  `![[moho-verde-01.jpg|400]]` seguido de un párrafo de caption.
- Todas las imágenes deben llevar `alt text` en el markdown para no romper
  `markdownlint` (regla MD045). Si se usa la sintaxis Obsidian pura, incluir
  también una descripción textual justo debajo.
