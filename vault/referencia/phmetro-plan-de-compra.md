# pHmetro — estado actual y plan de compra

## Estado actual

**Hoy no fermentamos.** El flujo del taller es **curado largo a ~15 °C,
HR 78–82 %, sin cultivo iniciador ni fase de estufado** (ver
[[Procedimiento-embutido]]).

Las barreras microbiológicas activas son:

- **Nitrito ingoing ≤ 80 ppm** — antibotulínico.
- **Deshidratación**: merma ≥ 30 % del peso fresco → a_w < 0,90.

En este flujo el **pH final no se controla** como PPC porque no hay
fermentación dirigida. No tener pHmetro, hoy, no bloquea la producción.

## Cuando se compre pHmetro

El repo evoluciona: se reactiva la fase de fermentación controlada
(cultivo iniciador + estufado) y se re-introducen como PPC el
**pH ≤ 5,3** y el presupuesto de **≤ 370 °C·h**. Marco técnico ya
documentado en [[fermentacion-referencia]] y [[barreras-control]].

## Recomendaciones de compra

Requisito imprescindible: **electrodo de punción (spear tip)** apto para
sólidos cárnicos. Los electrodos de vidrio para líquidos no sirven —
rompen y no penetran.

### Opciones

- **Hanna HI98150** (o similar de la gama HI98x) — pHmetro portátil con
  electrodo de punción para carnes, ATC (compensación automática de
  temperatura), calibración 2/3 puntos. Referencia del sector.
- **Milwaukee MW102** — alternativa más económica, calibración 2 puntos,
  ATC. Requiere adquirir aparte electrodo de punción compatible.
- **Apera PH60** — portátil con electrodos intercambiables; verificar
  compatibilidad con punta de punción para carne.

### Accesorios obligatorios

- Electrodo de **punción (spear tip)** para carne, si no viene incluido.
- **Buffers de calibración pH 4,01 y pH 7,01** (sobres o botella).
- Solución de conservación del electrodo (KCl 3 M); nunca guardar seco.
- Paño suave y agua destilada para enjuague entre mediciones.
